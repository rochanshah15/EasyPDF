PDF Manipulation App - User Documentation

Welcome to the PDF Manipulation App! This application allows you to perform various operations on PDF files, such as merging, splitting, extracting text/images/links, encrypting, decrypting, rearranging pages, rotating pages, adding metadata, creating PDFs, optimizing PDFs, and converting PDFs to Word and Excel documents. Below is a detailed guide on how to use the app effectively.

Getting Started

  1. Access the App: Open your web browser and navigate to the app's URL
  2. Upload Files: Ensure your PDF files are ready for upload. The app supports .pdf files only.
  3. Download Results: After performing an operation, the app will provide a downloadable file.

Features and Usage

Merge PDFs

Purpose: Combine multiple PDF files into a single PDF.

Steps:
  1. Go to the Merge section.
  2. Upload at least two PDF files using the file picker.
  3. Click Merge.
  4. Download the merged PDF file.

Split PDF

Purpose: Split a PDF file into individual pages.

Steps:
  1. Go to the Split section.
  2. Upload a PDF file.
  3. Click Split.
  4. Download a ZIP file containing all the split pages as separate PDFs.

Extract Text

Purpose: Extract text from a PDF file and save it as a .txt file.

Steps:
  1. Go to the Extract Text section.
  2. Upload a PDF file.
  3. Click Extract Text.
  4. Download the extracted text file.

Extract Images

Purpose: Extract images from a PDF file and save them as a new PDF.

Steps:
  1. Go to the Extract Images section.
  2. Upload a PDF file.
  3. Click Extract Images.
  4. Download the PDF containing all extracted images.

Extract Links

Purpose: Extract all hyperlinks from a PDF file and save them as a .txt file.

Steps:
  1. Go to the Extract Links section.
  2. Upload a PDF file.
  3. Click Extract Links.
  4. Download the text file containing all links.

Encrypt PDF

Purpose: Add a password to a PDF file to restrict access.

Steps:
  1. Go to the Encrypt section.
  2. Upload a PDF file.
  3. Enter a password in the provided field.
  4. Click Encrypt.
  5. Download the encrypted PDF file.

Decrypt PDF

Purpose: Remove password protection from a PDF file.

Steps:
  1. Go to the Decrypt section.
  2. Upload a password-protected PDF file.
  3. Enter the password in the provided field.
  4. Click Decrypt.
  5. Download the decrypted PDF file.

Rearrange Pages

Purpose: Reorder pages in a PDF file.

Steps:
  1. Go to the Rearrange section.
  2. Upload a PDF file.
  3. Enter the desired page order (e.g., 3,1,2).
  4. Click Rearrange.
  5. Download the rearranged PDF file.

Rotate Pages

Purpose: Rotate all pages in a PDF file by a specified angle (90°, 180°, or 270°).

Steps:
  1. Go to the Rotate section.
  2. Upload a PDF file.
  3. Select the rotation angle.
  4. Click Rotate.
  5. Download the rotated PDF file.

Add Metadata

Purpose: Add or modify metadata (title, author) in a PDF file.

Steps:
  1. Go to the Add Metadata section.
  2. Upload a PDF file.
  3. Enter the title and author in the provided fields.
  4. Click Add Metadata.
  5. Download the updated PDF file.

Read Metadata

Purpose: View the metadata (title, author, etc.) of a PDF file.

Steps:
  1. Go to the Read Metadata section.
  2. Upload a PDF file.
  3. Click Read Metadata.
  4. View the metadata on the results page.

Create PDF

Purpose: Create a new PDF from text and/or an image.

Steps:
  1. Go to the Create PDF section.
  2. Enter text in the text box.
  3. (Optional) Upload an image.
  4. Click Create PDF.
  5. Download the newly created PDF file.

Compress PDF

Purpose: Reduce the file size of a PDF.

Steps:
  1. Go to the Compress section.
  2. Upload a PDF file.
  3. Click Compress
  4. Download the compressed PDF file.

Convert PDF to Word

Purpose: Convert a PDF file to a Word document (.docx).

Steps:
  1. Go to the Convert PDF to Word section.
  2. Upload a PDF file.
  3. Click Convert.
  4. Download the converted Word document.

Convert PDF to Excel

Purpose: Convert a PDF file to an Excel document (.xlsx). Extract mainly tables.

Steps:
  1. Go to the Convert PDF to Excel section.
  2. Upload a PDF file.
  3. Click Convert
  4. Download the converted Excel document.

Troubleshooting

  • File Not Uploading: Ensure the file is in PDF format strictly
  • Invalid Password: Double-check the password entered for decryption.
  • No Text/Images Found: The PDF may be scanned or contain no selectable text/images.
  • Conversion Errors: If the PDF is scanned, the app will use OCR to extract text, which may take longer.

FAQs

  1. Can I merge more than two PDFs?

    Yes, you can merge as many PDFs as you want. But currently only 2.

  2. What happens if I upload a scanned PDF?

    The app will use OCR to extract text or images, but the accuracy depends on the quality of the scan.

  3. Is there a file size limit?

    The app does not impose a strict limit, but larger files may take longer to process.

  4. Can I rotate specific pages instead of all pages?

    Currently, the app rotates all pages in the PDF. For selective rotation, you can split the PDF, rotate individual pages, and then merge them back.

  5. How secure is the encryption?

    The app uses standard PDF encryption, which is secure for most use cases.
